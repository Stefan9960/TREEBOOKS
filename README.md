## 📌 About The Project

This project is a fully responsive landing page built using HTML5 and CSS3.

The main purpose of the project was to practice semantic HTML structure, layout building with Flexbox, and responsive design using Media Queries. The design adapts smoothly to mobile, tablet, and desktop screen sizes.

## 🛠 Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
* Responsive Web Design principles

## 📱 Features

* Fully responsive layout
* Mobile-friendly design
* Clean and structured semantic HTML
* Modern UI styling
* Organized file structure
